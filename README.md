# gameRobot
## 简介

gameRobot是一个主要用于游戏压测的机器人程序,理论上也可以用于其他.除了可以完成按模块的一应一答消息处理,还提供基于行为树的ai支持,让压测可以更接近玩家操作,另外也提供了对消息响应监控,以及可以自定义扩展的监控项目.
